# OficinaAPI
-- Parte da API do projeto OficinaAPI que tem como objetivo gerenciar uma oficina mecânica. 
-- A Api ira gerenciar os clientes, veículos, serviços e ordens de serviço.
-- A API foi desenvolvida em .NET Core 3.1 e utiliza o Entity Framework Core para persistência de dados.
-- A API foi desenvolvida utilizando o padrão de arquitetura DDD (Domain Driven Design).
-- A API foi desenvolvida utilizando o padrão de arquitetura REST.
-- A API foi desenvolvida utilizando o padrão de arquitetura SOLID.
-- A API foi desenvolvida utilizando o padrão de arquitetura CQRS (Command Query Responsibility Segregation).
-- A API é a parte de back-end do projeto OficinaAPI, a parte de front-end do projeto pode ser encontrada no repositório OficinaApp.
-- O front end será em Windows Forms e será desenvolvido em C#.
